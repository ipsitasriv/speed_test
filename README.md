# Typing Speed Test Game

A responsive Typing Speed Test web application built using **HTML**, **CSS**, and **JavaScript**. The application helps users improve their typing speed and accuracy by providing random paragraphs, live performance metrics, and an interactive user interface.

## Features

- Random typing paragraphs
- 60-second countdown timer
- Real-time Words Per Minute (WPM)
- Live Accuracy calculation
- Mistake counter
- Characters Per Minute (CPM)
- Restart test functionality
- Generate a new paragraph
- Responsive design for desktop and mobile
- High score stored using Local Storage
- Modern and clean UI

## How to Play

1. Open the application in your browser.
2. Click inside the typing area.
3. Start typing the displayed paragraph.
4. The timer starts automatically after your first keystroke.
5. Complete the paragraph before the timer ends.
6. View your final statistics including WPM, accuracy, mistakes, and CPM.
7. Click **Restart** to play again or generate a new paragraph.

## Performance Metrics

The application tracks:

- Words Per Minute (WPM)
- Characters Per Minute (CPM)
- Accuracy Percentage
- Number of Mistakes
- Remaining Time
- Highest Score
